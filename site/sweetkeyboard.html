Install SweetKeyboard

  1. Install SweetKeyboard from the App Store.
  2. Open the iPhone Settings app.
  3. Go to General > Keyboard > Keyboards.
  4. Tap Add New Keyboard.
  5. Select SweetKeyboard.
  6. Open any text field, tap the globe button, and choose SweetKeyboard.

Enable Full Access for clipboard tools

  1. Open iPhone Settings.
  2. Go to General > Keyboard > Keyboards > SweetKeyboard.
  3. Turn on Allow Full Access.
  4. Tap Allow when iOS asks for confirmation.
  5. Return to SweetKeyboard to review the available settings.

Full Access is optional. Basic typing, numbers, symbols, emoji, auto-capitalization, accents, and cursor controls can work without Full Access. Full Access is used for optional clipboard features and for sharing selected local settings between the app and the keyboard extension.

Settings

Settings explained

The SweetKeyboard app and the in-keyboard settings panel show the same main controls. On this page, the controls are informational only.

Clipboard Toolbar

Shows a toolbar above the keyboard with Copy, Clear Text Field, native iPhone Clipboard actions, SweetKeyboard Clipboard, Settings, and Hide Keyboard. Clipboard Toolbar requires Full Access.

Open Clipboard After Copy

After SweetKeyboard Copy saves selected text, SweetKeyboard can automatically open the local SweetKeyboard Clipboard so you can review or reuse saved snippets.

Native iPhone Clipboard Buttons

These rounded-square buttons appear when iOS reports plain text in the native iPhone Clipboard. You can choose which actions SweetKeyboard shows:

  • Import and Paste: import native iPhone Clipboard text into SweetKeyboard Clipboard and paste it into the active field.
  • Just Import: save native iPhone Clipboard text into SweetKeyboard Clipboard without pasting it immediately.
  • Just Paste: paste native iPhone Clipboard text into the active field without saving it into SweetKeyboard Clipboard.

Auto-capitalization

Automatically enables Shift at the start of compatible fields, after sentence-ending punctuation followed by a space, and after new lines. Email, URL, and username-style fields disable auto-capitalization.

Key Haptics

Adds light haptic feedback to supported keys and actions when the device supports it.

Swipe Cursor

Lets you swipe horizontally anywhere across the keyboard to move the cursor through text. Faster swipes move farther.

Forward Delete with Shift

When this option is enabled, tapping Shift once manually can make Delete remove the character after the cursor instead of the character before it.

Symbol Lock

Symbol Lock is controlled from the symbols and emoji keyboard row. When it is off, SweetKeyboard returns to letters after one symbol or emoji. When it is on, SweetKeyboard stays in the symbols or emoji layout for repeated entry.

Features

What SweetKeyboard does

Numbers stay visible

The main QWERTY keyboard keeps 1 through 0 visible for dates, addresses, codes, and passwords.

Symbols in one place

Native-style symbols are grouped on one symbols page, with punctuation and actions nearby.

Cursor keys

Dedicated left and right cursor keys help with precise edits without relying only on the iOS magnifier.

Swipe cursor movement

Horizontal swipes across the keyboard move the cursor, with speed-aware movement for longer edits.

Clear Text Field

The toolbar can clear the active field. Short fields usually clear immediately; long documents depend on the text context exposed by the host app.

SweetKeyboard Clipboard

Save copied text locally, keep up to 50 items, pin favorites above history, and paste saved snippets with one tap.

More built-in behavior

  • English QWERTY layout with a permanent top number row.
  • Bottom-row period key in letters mode.
  • Direct @ shortcut in email fields.
  • Emoji view available from the symbols layout.
  • Contextual action key for Return, Search, Go, Next, Send, Done, and related host actions.
  • Long-press accents for supported letters: a, c, e, i, n, o, u, and y.
  • Long-press period shortcuts such as ..., :, @, !, ?, and comma.
  • Fast typing support that keeps overlapping key presses in press order.

Info

Privacy, limits, and contact

Privacy

  • SweetKeyboard does not use analytics, advertising SDKs, tracking SDKs, cloud sync, or remote text processing.
  • Typed text is not uploaded by SweetKeyboard.
  • SweetKeyboard Clipboard data stays on your device.
  • Native iPhone Clipboard text is read only after you tap a native iPhone Clipboard action inside the keyboard.

Full Access

iOS shows a broad warning for all third-party keyboards that request Full Access. In SweetKeyboard, Full Access is used for clipboard tools and local app-to-keyboard sharing. SweetKeyboard does not use Full Access to upload keystrokes or clipboard content.

iOS limitations

  • Third-party keyboards are unavailable in some secure fields, including many password fields.
  • Some apps limit or block custom keyboards.
  • The Copy action depends on selected text exposed by the host app.
  • Clear Text Field is best effort in long documents because iOS may expose only part of the active text to a keyboard extension.
